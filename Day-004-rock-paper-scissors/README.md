## Features

- Beautiful ASCII art for Rock, Paper, and Scissors
- Random computer opponent
- Simple and intuitive console interface
- Clear win/lose/draw messages

## How to Play

1. Run the Python script
2. Choose your move:
   - Type `0` → Rock
   - Type `1` → Paper
   - Type `2` → Scissors
3. The computer will randomly pick its move
4. The winner is displayed instantly!
